# Ejercicio: Hello World

*[Read in English](README.md) | [Resolver en Línea](https://github.dev/matcom/gym/blob/main/exercises/hello_world/solution.py)*

## Antecedentes/Motivación

En la larga historia de la informática, el programa "Hello World" (Hola Mundo) es el primer paso tradicional para cualquiera que esté aprendiendo un nuevo lenguaje o explorando un nuevo entorno. Su propósito es simple: verificar que puedes escribir código correctamente, ejecutarlo y ver el resultado esperado. En **The Gymnasium**, este ejercicio también asegura que entiendas cómo usar la bandera `SUBMIT`.

## La Tarea

Implementa una función `hello_world()` que devuelva exactamente la cadena `"Hello, World!"`.

### Especificaciones

- **Nombre de la Función**: `hello_world`
- **Argumentos**: Ninguno
- **Tipo de Retorno**: `str`
- **Resultado Esperado**: `"Hello, World!"`

### Restricciones

El resultado debe coincidir exactamente, incluyendo mayúsculas y puntuación. No debe haber espacios en blanco adicionales.

### Ejemplo

```python
>>> hello_world()
'Hello, World!'
```

## Instrucciones

1. Abre `exercises/hello_world/solution.py`.
2. Implementa la función `hello_world`.
3. Cambia `SUBMIT = False` a `SUBMIT = True` en la parte superior del archivo cuando estés listo para ser evaluado.
4. Ejecuta `python solution.py` localmente para verificar tu solución con las pruebas integradas.
