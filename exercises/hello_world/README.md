# Exercise: Hello World

Welcome to **The Gymnasium**! 

In the long history of computer science, the "Hello World" program is the traditional first step for anyone learning a new language or exploring a new environment. Its purpose is simple: verify that you can correctly write code, run it, and see the expected output.

## The Task

Implement a function `hello_world()` that returns the exact string `"Hello, World!"`.

### Specifications

- **Function Name**: `hello_world`
- **Arguments**: None
- **Return Type**: `str`
- **Expected Output**: `"Hello, World!"`

### Example

```python
>>> hello_world()
'Hello, World!'
```
